# 削るべき語句のカタログ

中身を足さず「ちゃんと書いている感」だけを付ける語句。原則、削るか、具体的な事実・行動・数字に置き換える。

> 例文は「悪い例 → 良い例」。良い例が空欄のものは「丸ごと削除」が正解。

---

## 空虚な形容語（中身を名指さない修飾）

「重要な」「本質的な」「さまざまな」などは、何が・どう重要なのか、具体的に何があるのかを名指さないまま文を膨らませる。中身を具体で示せば、修飾語は要らなくなる。

代表例：**重要な／本質的な／根本的な／多角的な／包括的な／総合的な／画期的な／革新的な／最適な／さまざまな／幅広い／きめ細やかな／先進的な／次世代型の／柔軟な体制**

| 悪い例 | 良い例 |
|--------|--------|
| 本施策は事業成長において非常に重要な意味を持ちます。 | 本施策で来期の解約率を3%下げられます。 |
| さまざまな課題に対応してまいります。 | 納期遅延と在庫切れの2点に対応します。 |
| 包括的かつ多角的な観点から検討した結果 | コスト・納期・品質の3点から検討した結果 |
| お客様にとって最適なご提案をいたします。 | 御社の出荷量だと、月額プランより従量課金が2割安くなります。 |
| 根本的な解決に向けて取り組みます。 | 発注システムを月内に入れ替えて、二重発注をなくします。 |
| 幅広いニーズにお応えできる柔軟な体制を整えております。 | 英語・中国語の問い合わせにも、平日9〜18時で対応できます。 |
| 先進的な技術を活用した次世代型のプラットフォームです。 | 問い合わせ内容を自動で分類し、担当部署へ振り分けるツールです。 |

「名指せない」のは、中身が曖昧だという証拠。そのときは修飾語を削るより前に、何を言いたいのかを詰める。

## 空虚な動詞・流行り言い回し

行為の名前を出すだけで中身に踏み込まない動詞。「深掘りします」と言って終われば、何も深掘りされていない。

代表例：**向き合う／真正面から向き合う／寄り添う／掘り下げる・深掘りする／言語化する／腹落ちする／耳を傾ける／〜に資する／触れる・言及する**

| 悪い例 | 良い例 |
|--------|--------|
| お客様の課題に真摯に向き合ってまいります。 | いただいた不具合の報告に、48時間以内に返信します。 |
| 現場に寄り添ったサポートを心がけています。 | 導入初週は担当が現場に出向き、操作を一緒に確認します。 |
| 課題を深く掘り下げて分析しました。 | 過去6か月の問い合わせログを分類し、原因の上位3件を特定しました。 |
| チームの目指す方向性をしっかり言語化しました。 | 今期の目標を「解約率5%以下」と数値で定めました。 |
| この課題は次回の打ち合わせで深掘りしたいと思います。 | この課題は次回の打ち合わせで、原因の切り分けまで進めます。 |
| 懸念点について少し触れておきますと、納期が読みにくい状況です。 | 懸念点は納期です。部材の入荷が遅れており、現時点で確定できません。 |

## 空虚な強調・副詞

強調しているふりをするだけで情報を足さない。削っても意味が変わらないなら不要。強調語は、たいてい数字に負ける。

代表例：**非常に／極めて／大いに／まさに／いわば／ある意味／しっかりと／きちんと／徹底的に／改めて（空虚なとき）／圧倒的に／劇的に／間違いなく／言うまでもなく**

| 悪い例 | 良い例 |
|--------|--------|
| 非常に大きな成果を上げることができました。 | 売上が前年比1.4倍になりました。 |
| しっかりと対応させていただきます。 | 明日の午前中に修正版をお送りします。 |
| きちんと管理体制を整えております。 | アクセス権限を月1回見直し、退職者のIDは即日削除しています。 |
| 極めて高い品質を維持しています。 | 不良率は0.1%以下を維持しています。 |
| まさにこれこそが私たちの強みです。 | 納期を最短で翌日にできる点が、競合にはない強みです。 |
| いわば縁の下の力持ち的な存在です。 | 請求処理を裏で自動化し、経理の月末作業を3時間減らします。 |
| ある意味、今回の失敗は必然だったと言えます。 | テスト工程を省いたことが、今回の不具合の直接の原因です。 |

## 空虚な前置き（throat-clearing）

本題に入る前の空ぶかし。本当に結論を先に書くなら宣言は不要で、いきなり結論を書けばよい。「結論から言うと」と言って結論がないのは最悪のパターン。

代表例：**結論から申し上げますと／まず前提として／そもそもの話になりますが／言うまでもないことですが／本メールでは〜についてご連絡いたします／この件に関して申し上げますと／まず初めに〜について説明させていただきます／正面から〜する**

| 悪い例 | 良い例 |
|--------|--------|
| 結論から申し上げますと、この案で進めるのが良いと考えます。 | この案で進めましょう。 |
| 本メールでは、新システムの利用開始時期についてご連絡いたします。新システムの利用開始は7月1日です。 | 新システムの利用開始は7月1日です。 |
| まず前提としてお伝えしておきたいのですが、納期には余裕がありません。 | 納期に余裕がありません。あと3日です。 |
| 言うまでもないことですが、品質管理は重要です。 | 出荷前の検品を、現状の抜き取りから全件に変えたいです。 |
| この件については、正面から対応してまいります。 | この件は、来週中に原因を特定して再発防止策をまとめます。 |
| ご質問の件ですが、結論としては問題ございません。 | 問題ございません。（返信なら相手は何の話か分かっている） |

## 予告・総括の定型句

これから言う／今言ったことを宣言するだけで、中身を一切足さない枠取り。LLM が最も多用するパターン。主張があるなら、そのまま書けばよい。

代表例：**重要なのは〜という点です／ここでは〜について見ていきます／まとめると／要するに（直前の言い換えだけのとき）／〜ということになります／〜に他ならない**

| 悪い例 | 良い例 |
|--------|--------|
| 重要なのは、提出期限が今週金曜日であるという点です。 | 提出期限は今週金曜日です。 |
| まとめると、要するに、今回の対応で問題は解消されたということになります。 | 今回の対応で問題は解消されました。 |
| ここでは、今後のスケジュールについて見ていきたいと思います。 | 今後のスケジュールは以下のとおりです。 |

※「要するに」「まとめると」は、言い換えで情報が **増える**ときだけ使ってよい。同じことを繰り返すだけなら削る。

## 自己言及・締めの定型句

ブログ記事やAI出力の型をそのまま貼った締め。ビジネス文書では中身がなく、馴れ馴れしく響くことも多い。

代表例：**いかがでしたでしょうか／以上、〜について説明しました／ぜひ参考にしてみてください／少しでもお役に立てれば幸いです／最後までお読みいただきありがとうございました／何なりとお申し付けください**

| 悪い例 | 良い例 |
|--------|--------|
| いかがでしたでしょうか。本記事が参考になれば幸いです。 | （削除） |
| 以上、新機能について説明させていただきました。 | （削除。説明した事実を繰り返すだけ） |
| ぜひこの機会にご検討いただければと思います。 | ご検討のうえ、20日までにご返答いただけますか。 |
| 少しでもお役に立てれば幸いです。 | 不明点があれば、この返信にそのまま返してください。 |
| ご不明な点がございましたら、何なりとお申し付けください。 | 見積りの内訳で分かりにくい箇所があれば教えてください。 |

> **残す**：社外メールで「ご不明な点があればお知らせください」程度の簡潔な締めは正当な作法。削るのは「何なりとお申し付けください」のように空虚で大げさな版や、記事テンプレの名残（「いかがでしたか」「最後までお読みいただき」）だけ。

## 過剰な一般化・逃げの結論

「人によって異なります」「ケースバイケースです」だけで終わる文は、問いに答えていない。判断を避けて文を閉じる、典型的な逃げ。分かれ目の **基準**を示す。

代表例：**ケースバイケースです／人によって異なります／一概には言えません／一長一短があります／状況に応じて柔軟に判断する必要があります**

| 悪い例 | 良い例 |
|--------|--------|
| どちらが良いかは、ケースバイケースです。 | 月100件以上ならA案、それ未満ならB案が安くなります。 |
| 効果には個人差があり、一概には言えません。 | 既存ユーザーの平均では、導入後3か月で問い合わせが2割減っています。 |
| 状況に応じて柔軟に判断する必要があります。 | 在庫が10個を切ったら自動発注、それ以外は手動でお願いします。 |
| 一長一短があるため、慎重な検討が求められます。 | A案は安いが納期が長く、B案は高いが翌日納品です。予算優先ならA案を推します。 |

## 水増しの対句・四字熟語的並列

語呂が良いだけで具体性がない対句。2語並べても、情報は1語分も増えていない。

代表例：**迅速かつ丁寧に／柔軟かつ的確に／安心・安全／誠心誠意／スピーディーかつスマートに／高品質・低価格**

| 悪い例 | 良い例 |
|--------|--------|
| 迅速かつ丁寧に対応いたします。 | 当日中に一次回答し、原因と対策を明記して返します。 |
| 安心・安全なサービスを提供します。 | 通信は暗号化し、データは国内サーバーに保管しています。 |
| 誠心誠意、全力で取り組んでまいります。 | 納期に間に合うよう、今週は2名増員して進めます。 |
| 高品質・低価格を実現しています。 | 同等品質で他社より2割安く、不良率は0.1%以下です。 |

## 中身のない決意表明・未来志向の常套句

何を・どこまで・いつまでにが抜けた決意の型。謝罪文や報告の締めで多用される空文句。**お詫び文では、これを具体的な再発防止策に置き換えるのが最重要**（→ `references/registers.md`）。

代表例：**より一層努めてまいります／さらなる向上を目指します／邁進してまいります／精進してまいります／全力で取り組みます／所存でございます／善処いたします／真摯に受け止めます／鋭意対応中です／最大限の努力をしてまいります**

| 悪い例 | 良い例 |
|--------|--------|
| 今後はより一層、サービスの向上に努めてまいります。 | 再発防止のため、来月から出荷前の二重チェックを始めます。 |
| さらなる成長を目指し、邁進してまいります。 | 来期は新規顧客を月20社増やすことを目標にします。 |
| いただいたご要望につきましては、前向きに善処させていただきます。 | ご要望の納期短縮について、今週金曜までに対応可否をお返事します。 |
| このような事態を重く受け止め、再発防止に全力で取り組みます。 | 原因は手順書の不備でした。手順書を改訂し、来週全員に再研修します。 |
| スピード感を持って鋭意対応させていただいております。 | 昨日ご連絡の不具合は、本日15時までに修正版をリリース予定です。 |

## 読者への過剰配慮・許可・安心の乱発

根拠なく安心させようとして、逆に軽く響く。何をもって安心なのかが書かれていないと空虚。

代表例：**ご安心ください／大丈夫です／問題ありません／何の問題もございません／ご心配なく／焦らなくて大丈夫です**

| 悪い例 | 良い例 |
|--------|--------|
| ご安心ください。すべて問題なく進んでおります。 | 進捗は予定どおりです。設計が完了し、来週から実装に入ります。 |
| 大丈夫です、こちらで対応しておきますのでご心配なく。 | こちらで対応します。完了したら明日連絡します。 |
| 焦らなくて大丈夫ですので、ゆっくりご検討ください。 | 返答は来週月曜までで構いません。 |
| ご安心いただいて問題ございません。万全の体制で臨みます。 | 当日は私を含め3名で対応し、バックアップ機材も用意します。 |

> **残す**：具体的な根拠・対応とセットの「ご安心ください」は、相手の不安を鎮める正当な配慮。お詫び・クレーム対応では有効（例：「代替品を本日発送しました。明日中に届きますので、ご安心ください」）。削るのは根拠のない安心の乱発だけ。
