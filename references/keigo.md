# 敬語の slop と、残すべき敬語の線引き

AIが「丁寧にしよう」として作る典型的な失敗——二重敬語、「させていただく」濫用、クッション言葉の連打、慇懃無礼、謙譲・尊敬の取り違え。

**このファイルの目的は、敬語を一律に削ることではない。** 過剰・誤用・機械的な部分だけを直し、場面にふさわしい正しい敬語は残す。まず「残すもの」から確認する。

---

## 残すべき敬語・配慮（これは slop ではない）

一律に削ると、無礼・不自然になる。

| 残すもの | 理由 | 例 |
|----------|------|----|
| 冒頭の定型挨拶（社外宛て、1回） | 関係を円滑にする社外メールの慣習。省くと不躾。 | いつもお世話になっております。〇〇社の田中です。 |
| 宛名と名乗り | 誰から誰宛てかを示す基本作法。 | 株式会社ABC 営業部 佐藤様 |
| 依頼前のクッション言葉（1つ） | 依頼・断り・催促を和らげる正当な配慮。欠けると無礼。 | お手数ですが、ご確認のうえご返信いただけますか。 |
| 正しい尊敬語・謙譲語（一段で） | 相手や場面にふさわしい敬意。二重でなく一段なら正規。 | 明日10時に伺います。資料は拝見しました。 |
| 恩恵・許可が実在する「させていただく」 | 相手の許可・都合に依存する行為では正用。 | ご厚意に甘え、同席させていただきます。 |
| お詫びの誠実なへりくだり（簡潔に） | クレーム対応では丁重さが信頼回復に必要。 | このたびは多大なご迷惑をおかけし、誠に申し訳ございません。 |
| 慣用化した美化語（お／ご） | 「お願い」「ご連絡」「お客様」など定着した語の付加は正用。 | ご連絡ありがとうございます。 |
| 結びの定型（1回） | 社外メールの標準的な締め。 | ご確認のほど、よろしくお願いいたします。 |
| 相手の立場に応じた尊敬語の使い分け | 相手が目上か社外か身内かで形が変わるのが正しい。 | 部長がそうおっしゃっていました。 |

**register で適正量が変わる**：社外フォーマルメールでは上記をきちんと使う。社内Slackでは過剰になりやすく、軽い丁寧語で十分（→ `references/registers.md`）。

## 二重敬語（敬語の重ね過ぎ）

尊敬語・謙譲語を二重三重に重ねるのは、丁寧さではなく **誤用**。正しい敬語は一段で足りる。

| 悪い例（二重敬語） | 良い例 |
|--------|--------|
| 明日、御社へお伺いさせていただきます。 | 明日、御社へ伺います。 |
| 資料を拝見させていただきました。 | 資料を拝見しました。 |
| 部長がそのようにおっしゃられていました。 | 部長がそのようにおっしゃっていました。 |
| 社長がお見えになられました。 | 社長がお見えになりました。 |
| こちらをご覧になられてください。 | こちらをご覧ください。 |
| お客様がお帰りになられました。 | お客様がお帰りになりました。 |

「伺う」「拝見する」「おっしゃる」「ご覧になる」などは、それ自体で敬語が完成している。さらに「お〜させていただく」「〜られる」を足さない。

## 「させていただく」の濫用

「させていただく」は本来〈相手の許可を得て／相手の恩恵を受けて〉行うときに使う。許可も恩恵も関係ない自分の行為すべてに付け、一文に何度も重ねると、空虚にへりくだって見え、回りくどく、自信のなさを印象づける。多くは「いたします」「します」で足りる。

| 悪い例 | 良い例 |
|--------|--------|
| 本日より新サービスを開始させていただきます。 | 本日より新サービスを開始いたします。 |
| 担当の田中とさせていただいております。 | 担当は田中でございます。 |
| 資料を添付させていただきます。 | 資料を添付します。 |
| 後ほど資料についてご説明させていただきます。 | 後ほど資料についてご説明します。 |
| こちらの商品は完売とさせていただきました。 | こちらの商品は完売いたしました。 |
| ご担当者様にお繋ぎさせていただきます。 | ご担当者様におつなぎいたします。 |
| 請求書を送付させていただきましたので、ご確認させていただけますでしょうか。 | 請求書をお送りしましたので、ご確認いただけますでしょうか。 |

**判別の目安**：その行為に相手の許可・恩恵が実在するか。実在しない（自社の判断・自分の行為）なら「いたします」。実在する（相手が許可した・相手の都合に乗る）なら「させていただく」を残す。相手の招待を断る、相手の予定に影響する中止・延期を伝えるなど、相手の厚意や都合が絡む場面では正用になりうるので、一律に削らない（断りの直し方は `references/registers.md`）。

## クッション言葉の過剰連打

「恐れ入りますが」「お手数をおかけしますが」「誠に勝手ながら」は、一通に一つか二つで効く。段落ごと・文ごとに置くと、本題がクッションに埋もれる。謝りすぎは誠意ではなくノイズ。

| 悪い例 | 良い例 |
|--------|--------|
| 恐れ入りますが、お手数をおかけして大変恐縮ですが、誠に勝手ながら、ご確認いただけますでしょうか。 | お手数ですが、ご確認いただけますでしょうか。 |
| 恐れ入りますが添付をご確認ください。恐れ入りますがご返信もお願いいたします。恐れ入りますが期限は明日です。 | 添付をご確認のうえ、明日までにご返信ください。 |
| お忙しいところ大変恐縮ではございますが、ご多忙の折、誠に申し訳ございませんが、少々お時間を頂戴できますと幸いです。 | お忙しいところ恐縮ですが、少しお時間をいただけますか。 |

## 慇懃無礼・回りくど過ぎる丁寧

丁寧語を積み上げすぎて、「何をしてほしいのか」「いつまでに」が読み取れない文。長くて低姿勢だが中身が薄い。要点を一文で言えず、敬語の装飾でかさ増ししている。

| 悪い例 | 良い例 |
|--------|--------|
| もし可能でございましたら、ご無理のない範囲で、お差し支えなければで構いませんので、ご都合のよろしいタイミングでご一報いただけますと大変ありがたく存じます次第でございます。 | ご都合のつくときに、ご一報いただけますか。 |
| このたびは、私どもの不手際によりまして、ご迷惑をおかけすることになってしまいましたことにつきまして、深くお詫び申し上げる次第でございまして、誠に申し訳ない気持ちでいっぱいでございます。 | このたびは私どもの不手際でご迷惑をおかけし、誠に申し訳ございません。 |
| 恐縮ではございますが、何卒ご検討のほどよろしくお願い申し上げたく存じ上げる所存でございます。 | 何卒ご検討のほどよろしくお願いいたします。 |
| ご返信につきましては、お手すきの際で全く問題ございませんので、お急ぎにならず、ごゆっくりで結構でございますので、いつでも構いません。 | ご返信はお手すきの際で結構です。 |

## 謙譲・尊敬の取り違え／自分側に尊敬語

自分や身内の動作に尊敬語を、相手の動作に謙譲語を当てる取り違え。文法的に誤りであるだけでなく、相手を下げ自分を上げる失礼になりうる。「誰を立てるか（方向）」を正す。

| 悪い例 | 良い例 |
|--------|--------|
| 私が御社へお越しいたします。 | 私が御社へ伺います。 |
| こちらの資料を拝見してください。 | こちらの資料をご覧ください。 |
| 弊社の社長がそうおっしゃっています。（社外向け） | 弊社の社長がそう申しております。 |
| 先ほど課長が申されていました。（謙譲語「申す」＋「れる」の混交誤用） | 先ほど課長がおっしゃっていました。 |
| あなた様のご意見を申し上げてください。 | ご意見をお聞かせください。 |
| 頂戴したご質問にお答えしてください。 | いただいたご質問にお答えします。 |

※社外向けでは自社の人間は「身内」なので、社長でも謙譲語「申す」。社内で目上を立てる場面では尊敬語「おっしゃる」。立場で変わる。

## 過剰な美化語・「お／ご」の付け過ぎ

名詞という名詞に「お」「ご」を付けて丁寧さを演出する。外来語や付けない慣習の語にまで付き、不自然・くどくなる。一文に「お／ご」が密集すると幼稚にすら見える。

| 悪い例 | 良い例 |
|--------|--------|
| お打ち合わせのおスケジュールとお資料をご準備いたします。 | 打ち合わせの日程と資料を準備いたします。 |
| ご担当者様のおメールアドレスをお教えください。 | ご担当者様のメールアドレスを教えていただけますか。 |
| お時間のあるときにお電話とお返事をお願いいたします。 | お手すきのときに、お電話かご返信をお願いいたします。 |

## 丁寧表現の冗長な語尾

「〜と存じます」「〜次第でございます」「〜のほどよろしく」を必要以上に連結し、語尾だけが長くなる。意味は増えず、形式だけが膨らむ。簡潔な敬体で十分。

| 悪い例 | 良い例 |
|--------|--------|
| ご対応いただけますよう、お願い申し上げる次第でございます。 | ご対応のほどよろしくお願いいたします。 |
| 確認が取れ次第、改めてご連絡を差し上げたく存じ上げます。 | 確認が取れ次第、改めてご連絡します。 |
| ご検討のほど何卒よろしくお願い申し上げたいと考えております所存でございます。 | ご検討のほど何卒よろしくお願いいたします。 |
| ご不明な点がございましたら、お気軽にお問い合わせいただけましたら幸いに存じる次第でございます。 | ご不明な点があれば、お気軽にお問い合わせください。 |
| ご確認させていただかせていただきたく存じます。 | ご確認いただけますでしょうか。 |
